## 📚 関連プロジェクト: IBMi-Outlook統合

IBMiとOutlook 2506をプライベートネットワーク経由で接続するための包括的な設計・実装ドキュメントを作成しました。

### ドキュメントの場所
ドキュメントは別リポジトリに移動しました：

**主要ドキュメント:**
- 基本実装計画書 - SMTP/POP3を使用した基本設計
- 認証対応策 - Microsoft認証方式変更への対応
- ILE-C実装計画 - OAuth 2.0プロキシの完全実装
- 接続検証機能 - 事前検証ツールの設計

### 特徴
- ✅ 中間サーバー不要（IBMi上で完結）
- ✅ OAuth 2.0完全対応
- ✅ ILE-Cによる高パフォーマンス実装
- ✅ 包括的な接続検証機能
- ✅ セキュアなトークン管理
